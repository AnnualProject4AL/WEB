# WEB

### Made in JEE with :
  - Spring Boot : https://projects.spring.io/spring-boot/
  - Thymeleaf : https://www.thymeleaf.org/
  - Junit : https://junit.org/junit5/
  - Maven : https://maven.apache.org/
  
### Command :
Build : 

```
$ mvn clean install
```

Generate Jar : 

```
$ mvn clean package
```

Launch test : 

```
$ mvn clean test
```

Launch the server and projet : 

```
$ mvn spring-boot:run
```
